## Setup

1. Use [Laravel Herd](https://herd.laravel.com/).

2. Clone this repository:
   ```
   git clone https://github.com/jessiesiat/ecommerce-app.git
   cd ecommerce-app
   ```

3. Install dependencies and set up the project:
   ```
   composer run setup
   ```

4. Seed the database:
   ```
   php artisan db:seed
   ```

5. Visit [http://ecommerce-app.test](http://ecommerce-app.test) in your browser.
